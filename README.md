# StayHub

숙박 예약 플랫폼

## 프로젝트 목적

AI 코딩 도구(Claude Code)를 활용한 개발 워크플로우를 직접 경험해보고 싶었습니다.
도메인 설계 검토, 코드 생성, 오류 디버깅까지 AI와 함께 진행하며
어떤 부분에서 효과적인지, 어디서 직접 판단이 필요한지를 몸으로 익히는 것이 목표였습니다.

## 기술 스택

**Backend**
Java 17 · Spring Boot · Spring Security · JPA · QueryDSL · MySQL · Redis · JWT

**Frontend**
Vue 3

**Infra**s
Docker · LocalStack(S3) · Toss Payments
